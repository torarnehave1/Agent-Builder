# Agent Builder

Visual builder for creating autonomous AI agents using the Claude Agent SDK and Vegvisr knowledge graph system.

## Purpose

Learn how to use AI Agents with the Vegvisr system by building a visual agent builder app that:
- Creates and configures AI agents
- Integrates with knowledge graphs
- Uses Claude Agent SDK primitives
- Provides a UI for agent management

## Included
- React 19 + TypeScript + Vite
- AuthBar + magic link login (cookie.vegvisr.org)
- EcosystemNav
- Language selector + i18n scaffold
- Tailwind v3 setup
- vegvisr-ui-kit integration

## Quick Start
```bash
npm install
npm run dev
```
